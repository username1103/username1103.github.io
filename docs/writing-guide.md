# 글쓰기 규칙

## 1. 단문 원칙
- 한 문장은 15~20 단어 이내로 작성한다.
- 한 문장에 주어와 서술어는 하나씩만 사용한다.

## 2. 간결성
- 불필요한 미사여구를 제거한다.
- 불필요한 내용은 삭제한다.

## 3. 중복 금지
- 중복 표현을 사용하지 않는다.
- 겹말을 피한다.

## 4. 두괄식 구성
- 핵심 내용을 먼저 쓴다.
- 결론을 앞에 배치한다.


# 글쓰기 흐름

## 1. 초안 작성하기
- 위 규칙에 맞춰 글 초안을 작성합니다.

## 2. 퇴고하기
- 작성한 초안에 대해 n번 퇴고를 진행하며 다듬어 나갑니다.

### 2-1. 구성 확인
- 정한 주제에서 벗어나는 이야기가 있는지 확인합니다.
- 빼거나 더하거나 순서를 바꿔야할 내용이 있는지 확인합니다.
- 각 단락이 매끄럽게 이어지는지 확인합니다.

### 2-2. 제목 확인
- 제목과 소제목의 내용이 일치하는지 확인합니다.
- 제목과 소제목이 내용을 짐작할 수 있을 만큼 구체적인지 확인합니다.

### 2-3. 문장 확인
- 사족, 반복되는 표현이 있는지 확인합니다.
- 맞춤법이나 오탈자가 있는지 확인합니다.
- 문장의 길이는 적당한지 확인합니다.
- 주어와 술어는 조화로운지 확인합니다.
  - 예시
    - AS-IS) 고객의 의견은 홈페이지의 홈 버튼 위치를 바꾸라고 합니다.
    - TO-BE 1) 고객이 홈페이지의 홈 버튼 위치를 바꾸라고 합니다.
    - TO-BE 2) 고객의 의견은 홈페이지의 홈 버튼 위치를 바꾸라는 겁니다.
